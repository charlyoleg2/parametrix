designix-ui
===========


Presentation
------------

*designix-ui* is a *svelte-kit* application for presenting the designs of *designix*.


Requirements
------------

- [node](https://nodejs.org) > 20.10.0
- [npm](https://docs.npmjs.com/cli) > 10.5.0


Getting started
---------------

```bash
git clone https://github.com/charlyoleg2/parametrix
cd parametrix
npm install
npm run ci
npm run preview
```
