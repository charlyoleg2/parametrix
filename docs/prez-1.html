Parametrix
CAD as code
Yes, but all CAD tools have an API!
Indeed, but 3 main advantages with Parametrix
1- Written in Modern Javascript
Run in CLI and Browser
Design easy to share via websites and packages
2- Durability of your 3D modeling
Parametrix is Open source and lightweight
(geometrix: 6k, geomui: 3k, geomcli: 1k)
3- Independance from CAD tools
Model first then choose your CAD tool
And other benefits
4- Documented parameters
5- Make easily your Web-UI and CLI
Just in few lines of code
6- 2D simulations
Examples for kinetic and optic simulations
7- API optimized for manufacturing
8- Extendable API via npm-package
API for metal sheet, API for architecture
