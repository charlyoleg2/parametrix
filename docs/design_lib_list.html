List of geometrix design libraries

Below a list of libraries of 3D-part-designs based on Geometrix:

  1. designix : src - ui - pkg : The first and reference design library. Geometrix features were selected to support those designs
  2. desi51 : src - ui - pkg : First batch of geometrix design examples for the tutorial
  3. desi52 : src - ui - pkg : Second batch of geometrix design examples for the tutorial
  4. desi53 : src - ui - pkg : Third batch of geometrix design examples for the tutorial
  5. desi53b : src - ui - pkg : Fourth batch of geometrix design examples for the tutorial
  6. desi78 : src - ui - pkg : The designs of Charlyoleg
  7. desi82 : src - ui - pkg : The designs of a friend
  8. desi15L : src - ui - pkg : The designs of a junior developer

If you want to get your geometrix design library in this list, please, create a Github issue with a link to your library.
